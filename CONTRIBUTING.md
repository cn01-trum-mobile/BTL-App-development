# 🤝 Quy tắc đóng góp

## 📂 Quy tắc branch
- `main` → code stable
- `dev` → code phát triển
- `feature/xyz` → nhánh cho từng tính năng
- `fix/bug-name` → nhánh fix bug

## 🔄 Quy trình PR
1. Tạo branch từ `dev`.
2. Commit rõ ràng, ví dụ: `feat: add login screen`.
3. Mở Pull Request → chọn reviewer.
4. Pass review + CI thì mới merge.
